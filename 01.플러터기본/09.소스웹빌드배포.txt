[ 플러터 원본 소스를 웹배포용 소스로 빌드하기 ]

1. 빌드할 플러터 프로젝트 위치로 이동한다.(cd...)

2. 플러터 명령어로 웹 빌드한다.
플러터 빌드 명령어 : 특정 플랫폼에 맞게 빌드
웹은 web!
- flutter build web

3. 빌드 후 작업하기

(1) build폴더 하위 web폴더아래 index.html 파일을 연다
  <base href="/">
    소스중 이 부분이 루트를 연결하는 부분이다
    만약 서버에 올리면 서버 루트 주소를 넣고 
    라이브 서버로 연습중이면 그 주소를 띄워보고 복사하여 넣어준다.

(2) 대부분이 출력되지만 이미지가 출력되지 않는 경우
    개발자 모드에서 이미지 404에러 경로를 보고 실제 경로가 맞지 않는 폴더를 이동하여
    이미지 경로를 맞춰준다
    (여기서는 assets/assets/images -> assets/images)
